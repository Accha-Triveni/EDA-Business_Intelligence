# EDA-Business_Intelligence
Performed Exploratory Data Analysis (EDA) and Business Intelligence on a Superstore dataset using Python, SQL, Pandas, Matplotlib, and Seaborn. Analyzed sales, profit, customer trends, and regional performance through statistical analysis, SQL queries, visualizations, and KPI dashboard creatio

## Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** and **Business Intelligence (BI)** using a Superstore sales dataset.
The main objective is to uncover:
* Sales patterns
* Customer behavior
* Profit trends
* Regional performance
* Product insights

The project also demonstrates:
* Data cleaning
* Statistical analysis
* SQL querying
* Data visualization
* Dashboard creation
